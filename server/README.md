```txt
    考试信息管理：{
        选择试卷库：
        发布：{
            确认一些信息
            考试说明
        }
    }
    试题库：{ // 批量， 手动，Excel
        分类：
        题型：{ // 题干，解析
            单选：选项(+)
            多选：选项(+)
            判断：T/F
            填空：Input(+)
            问答：Textarea(keyword)
            组合：
            录音：Textarea(转换音频)
        }    
        难度：简单，普通，困难
    }
    试卷库：{
        创建试卷：{ // 新建，从试卷库
            名称

            答题时长：整卷限制时长，单题限制时长
        }
        选题组卷：{
            选择题型题目，设置时间
        }
    }
    成绩查询批改?
    分析考试?
    分析考生?
``` 
